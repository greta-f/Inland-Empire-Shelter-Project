🐾 California Animal Shelter Crisis: Inland Empire 🐾

A web GIS project examining shelter distribution, county-level outcomes, and the animal shelter crisis in the Inland Empire

Dashboard Leaflet Map

Project Overview

This project maps animal shelter locations in the Inland Empire and compares county-level shelter statistics with broader state trends. The dashboard below focuses on Riverside and San Bernardino counties, while the Leaflet page provides an interactive web map depicting some Riverside and San Bernardino serving rescues.
